DISCLAIMER

  • It was monumentally awesome
  • Not standing still

Subversion:

Enterprise-class centralized version control for the masses

The Goodies

  • Code Snapshot History
  • Central Repository
  • Worldwide Collaboration
  • First-Class Directories

Why Do I Despise It So?

  • Slow
  • Big
  • Linear History
  • Central Repository
  • Branching Sucks
  • Merging Sucks More
  • Rename = Copy + Delete

git

DISCLAIMER

  • Something else will be monumentally awesome
  • Command Line Interface

git:

Free & Open Source, Distributed Version Control System

The Goodies

  • Code Snapshot History Graph
  • Central Repository Fully Distributed
  • Worldwide Collaboration Domination
  • First-Class Directories Branches

Why Do I Love It So?

  • Fast (like, really fast)
  • Small
  • Directed Graph History
  • Fully Distributed (fun workflows, auto-backup)
  • Branching Rocks
  • Powerful Merging
  • Incremental Commits

DISCLAIMER

  • Something else will be monumentally awesome

GitHub:

Social Network Where the First-Class Citizen is Code

The Goodies

  • git (and svn) hosting
  • Namespaced Repositories
  • Repository-controlled homepage and wiki
  • Fork Discovery
  • Intelligent Fork Pulling
  • Private Repositories

Why Do I Love It So?

  • Cool, Hip
  • Almost Friction-less
  • Code-Focused
  • No Ads
  • Open Source Made Easy
